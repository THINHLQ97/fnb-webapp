# F&B Admin Dashboard

Admin dashboard cho quản lý quán nước — chấm công, nhân sự, CMS, RBAC.

## Tech Stack

- Next.js 16 (App Router)
- Auth.js v5 (Credentials + Google OAuth)
- Prisma 6.x + PostgreSQL
- shadcn/ui + Tailwind CSS
- TanStack Query + TanStack Form

## Development

```bash
npm install
npm run dev
```

## Environment Variables

Copy `.env.example` → `.env` và điền thông tin.
